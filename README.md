| Category | Details |
|---|---|
| Stack | React, FastAPI, OpenCV, WebSockets, SQLite, OCR |

| Impact | Helps demonstrate AI-powered civic enforcement, road safety automation, and intelligent public infrastructure systems |
| Repository | [GitHub](https://github.com/Aryanbbc/AutoSentinel) |

It represents an end-to-end AI product pipeline from video input to actionable enforcement records.

The React dashboard provides a cinematic interface for metrics, logs, memory feeds, and skill execution, making it a strong AI systems engineering project.
